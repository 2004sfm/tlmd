# tldm (Terminal Lightweight Display Manager)

`tldm` es un Display Manager ultra-minimalista y rápido para la TUI, diseñado específicamente con una estética **True Black (`#000000`)**, optimizado para pantallas OLED y usuarios que trabajan desde la CLI.

## Características Principales

- **Minimalismo y Rendimiento:** Autenticación segura mediante PAM directo desde la TUI, sin demonios gráficos en segundo plano consumiendo recursos.
- **Diseño OLED-Friendly:** Interfaz completamente en True Black, evitando el cansancio visual y el burn-in en monitores OLED.
- **Integración Transparente:** Actúa como el puente perfecto entre la TTY nativa de Linux y gestores de sesión modernos (como UWSM).
- **Gestión de Cuentas:** Ideal para sistemas con múltiples cuentas de usuario que requieren un inicio de sesión rápido y seguro.

## Objetivo del Proyecto

Brindar un entorno de autenticación puro y eficiente que no interfiera con el rendimiento del sistema, ideal para quienes buscan mantener su instalación lo más ligera y controlada posible.

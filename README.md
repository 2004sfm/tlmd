# tldm (Terminal Lightweight Display Manager)

tldm es un gestor de inicio de sesión (Display Manager) para la TUI ultra-minimalista, rápido y diseñado bajo una estética True Black (`#000000`) ideal para pantallas OLED. Está pensado específicamente para entornos CLI y usuarios de tiling window managers que manejan múltiples cuentas en el sistema.

El proyecto actúa como el puente perfecto entre la TTY nativa de Linux y gestores de sesión modernos como UWSM, permitiendo autenticar usuarios de forma segura mediante PAM sin sacrificar rendimiento ni añadir demonios gráficos innecesarios en segundo plano.
